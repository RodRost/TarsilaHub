Supabase connection

Connect to your Supabase project so data persists in a real database — accessible from any device, by your whole team.

  1. Go to app.supabase.com → create a project (free tier is fine)
  2. Run supabase-setup.sql in the SQL Editor
  3. Go to Project Settings → API and copy the Project URL and anon key below
Found in Project Settings → API → Project URL

Claude AI proxy

Required for AI features (1:1 prep, staffing assistant, digest) to work from your public domain. The proxy is a Supabase Edge Function that forwards requests to Anthropic's API securely.

  1. Go to Supabase → Edge Functions → New function, name it claude-proxy
  2. Paste the Edge Function code (see the instructions above this settings panel)
  3. Go to Edge Functions → Manage secrets → add secret ANTHROPIC_API_KEY with your key from console.anthropic.com
  4. Deploy the function and paste its URL below
After saving, AI features will route through this URL instead of calling Anthropic directly.
